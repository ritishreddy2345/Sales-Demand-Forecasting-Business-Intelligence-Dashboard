# Project Summary

## Project Title

Sales Demand Forecasting & Business Intelligence Dashboard

---

## Overview

This project simulates a real-world retail analytics workflow by generating synthetic retail transaction data, performing customer segmentation, forecasting product demand, and producing dashboard-ready outputs for business intelligence reporting.

The project is built entirely using Python and demonstrates skills in ETL pipeline development, customer analytics, machine learning, forecasting, and dashboard planning.

---

## Objectives

* Generate realistic retail transaction data
* Build an automated ETL pipeline
* Perform customer segmentation using RFM analysis and K-Means clustering
* Forecast future demand using time-series forecasting models
* Create dashboard-ready datasets for business intelligence reporting
* Design a Looker Studio dashboard blueprint

## Project Workflow

### Phase 1 – Data Generation & ETL Pipeline

Synthetic retail transaction data is generated with realistic business characteristics.

Features include:

* 50,000+ transaction records
* 12 product categories
* 4,999+ customers
* Revenue calculations
* Date-based transaction history

Output:

output/transactions.csv

---

### Phase 2 – Customer Segmentation

Customer purchasing behaviour is analysed using:

* Recency
* Frequency
* Monetary Value (RFM)

K-Means clustering is applied to group customers into meaningful business segments.

Example segments:

* VIP Active
* High Value
* At Risk
* Standard Customers

Output:

output/customer_segments.csv

---

### Phase 3 – Demand Forecasting

Daily sales data is aggregated and forecasting models are applied.

Forecasting models:

* ARIMA
* Prophet (optional)

Forecasts are generated for:

* Revenue trends
* Product category demand
* Future business planning

Outputs:

output/daily_sales_aggregated.csv

output/forecast_results.csv

output/forecast_model_metrics.csv

---

### Phase 4 – Dashboard Blueprint

A complete Looker Studio dashboard blueprint is generated.

Dashboard Sections:

1. Revenue Trends & Forecasting
2. Inventory Risk Analysis
3. Customer Segmentation Insights

Output:

output/Looker_Studio_Complete_Blueprint.md
